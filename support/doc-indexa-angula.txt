> Projeto com Angular:
	https://cursos.alura.com.br/course/angular-aplicacao-web-componentes-linguagem-template-cli/task/147543
	Install:
		nodejs: sudo apt update /	sudo apt install nodejs
		Angular: npm install -g @angular/cli@17.0.3 (instala global)
		Cria projeto: ng new indexa (indexa é o nome do projeto a pasta que será criado)
	Run serve:
		sudo apt install ng-common (caso ng serve não rode de primeira)
		ng serve

> Rodar projeto (entrar na pasta que criou no caso esse indexa)
	Iniciar servidor: ng serve

> Criar componente
	ng generate component componentes/container
		> app.component.ts
			imports: novo componente
		> usar componente
			no app.component.html que é o componente principal eu uso 
			<app-container> conteúdo </app-container>
